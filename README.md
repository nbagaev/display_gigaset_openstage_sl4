# Gigaset OpenStage SL4 Display
## Arduino test sketch for Gigaset OpenStage SL4 Display
### Labeled as: WD-F1216YU-6FLWa
### Controller like ILI9163, ST7735? 
#### Pinout:
```
1  = +4,7V BG LED

2  = Control BG LED (PWM 2,5V)

3  = D/CX

4  = SCX

5  = RESX

6  = SDA

7  = VDDI (+1,8V)

8  = SCL

9  = GND

10 = VDD (+2,8V)
```
